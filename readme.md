# Ein eigenes Korpus erstellen - per Hand oder mit einfachen Tools

Dieses Mini-Tutorial zeigt an einem einfachen Beipsiel, wie man einerseits mit niedrigschwelligen Mitteln, andererseits mit etwas anspruchsvolleren computationalen NLP-Tools korpuslinguistisch arbeiten kann.
